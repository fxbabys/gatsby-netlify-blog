这几个月来面了很多次试，从去年十二月开始断断续续地投递简历到今年三月正式进入大厂的面试流程，如今我的春招实习之路算是结束了，期间面过不少小厂拿到 offer，也面过不少大厂最终进入等待 offer 流程，目前在选择的有腾讯、百度与滴滴，腾讯与百度都在等 offer call，滴滴拿到 offer，现在想把这段经历记录下来以供秋招包括以后可能的面试参考。

话不多说，开始吧~

按时间线开始：

### 头条

去年11月中投了一次，没反应...12月中又投了一次，还是没反应...后面加到一个在头条的江财学长说头条实习会比较卡学历，建议我秋招再试试，于是后面就放弃了。

### 英伟达

工作室学长直推他的部门，很快就有了面试邀约，简单问了些前端的问题加上一些数据库等等之后就过了...

### 日志易

在牛客上看到它的招聘，抱着试试看的心态（被福利吸引）然后就加了 hr，直接邀约了第二天的面试：

#### 一面

- CSS
  - 盒模型
  - BFC
  - 行内元素、块级元素
  - 一个弹框 display: none 加载时页面闪一下 怎么解决
  - 重绘与回流
- JS
  - 基本类型、引用类型
  - this
  - EventLoop
- Vue
  - 子组件怎么调用父组件的方法

#### 二面

- 主要根据简历、写过的博客提问：
  - 聊学习过程、培训、工作室、专业课学习等等
  - CSS 垂直居中实现
  - Vue、React 比较
  - WebSocket 原理
  - 除了 MVVM 还知道哪些设计模式
- 算法：
  - 斐波那契数列：递归、非递归、动态规划 --- 新思路：利用数学公式
  - 最大子段序列和：动态规划
  - 求质数：循环相除到一半即可 --- 利用数学公式

####三面

- 交流为主：
  - 聊工作室管理、项目进度、个人学习、技术发展等等
  - 了解了他们公司的技术栈、团队人员、实习会做什么

#### 四面

- 共享屏幕敲代码：
  - 判断一个括号串是否有效
  - 总共有 8 颗糖，每天吃 1 颗糖或 2 颗糖，共有几种吃法

### 京东

牛客上投的，本来以为是京东金融，之后发邮件约时间时发现已经是新成立的京东数科部门。

#### 一面

- 原型链 new Function 出来的对象 _ _proto_ _ 指向
- ES6 如何处理多个发出去的请求
- CSS 盒模型
- position 属性
- jq vs react vs vue

之后就投入到期末备考当中，然后是在家的面试

### 即刻

在掘金上看到招聘信息，同样抱着试试看的心态（被福利吸引）投了，一个礼拜后 hr 约我面试

#### 一面

- 主要问了我自己封装的两个框架信息
- 在线编程：实现一个柯里化函数

#### 二面

- 算法题（说思路）：
  - 两个子节点找最近公共祖先
  - 反转一个单链表
- 还是问了我自己封装的框架信息

即刻的面试感觉主要就是围绕我自己封装的框架来问，具体跟面试官沟通下来入职以后也是会一起做基建类的工作，这个还是很 nice 的，同时即刻的福利真的很好，强推！

### 滴滴

在牛客上看到招聘信息，组内直投

#### 一面

- 主要交流项目经验，问一些基础问题
- 考察了快排

#### 二面

- 也是主要交流项目经验，还问了我具体重构一个旧系统会怎么做
- 本来也想玩问我快排的，我说一面问了就改问归并排序了...

整体面试下来感觉主要需要的是有比较丰富项目经验的人，因为过去是需要重构一个比较老的系统

过了一周给了口头 offer，然后正式的就一直发不出来了（==）

### 百度

在某个群里看到，直接给了主管，然后很快就约了在线编程面试：

- 石墨文档 coding: 计算千位分隔符（这里主要是要考虑到极限情况，比如末尾不能有逗号）
- doctype
- iframe 优缺点，ajax 替代
- 多个标签页之间如何通信
- 项目讲解

之后就来学校了，第二天就开始准备好自己的春招：

### 腾讯

牛客上看到 AlloyTeam 的招聘，组内直推，来校后第一周就约了具体的面试

#### 一面

- 项目难点：hy-vue-admin、jeem React 如何渲染对应的组件？
- Vue 响应式原理、computed 实现、React diff
- Vue 和 React 理解对比、单向和双向数据流、直出（同构）了解吗？
- Node require 同步还是异步、循环引用问题
- 安全：XSS、CSRF
- HTTP/2、HTTPS
- 性能优化实践
- 霍夫曼编码

### CVTE

2.27 号做了笔试一直到 3 月中才约的牛客视频面试：

- 主要还是一些基础问题
- 两道在线编程
  - 计算 js 时分夹角
  - 给定一个长度为 m*n 的字符串，以逆时针螺旋顺序，输出为一个二维矩阵（剑指 offer 29）

面完了两个面试状态都显示通过，然后终面一直等到现在...

### 滴滴

在牛客上看到滴滴另一个部门又招实习，看 JD 与自己目前专注的还挺符合就投了过去，当天就约了一面：

#### 一面

- 基础题（HTML、CSS、JS）...

- 两道编程题

  - 输入：['2018-02-03', '2018-01-25', '2018-02-01'];输出：['2018-01-25', '2018-02-01', '2018-02-03']

  - ```js
    function Hero (name) {
    // 实现代码
    }
    var hero = new Hero('哈哈');
    hero.kill('2').blood('20');
    
    //输出：
    //英雄哈哈诞生
    //哈哈击杀了2个敌人；
    //哈哈造成了20点伤害；
    ```

#### 二面

- 基本都是基础面经题
- 了解了工作室的情况

#### 三面

- 问了一些框架原理性问题

然后确认完入职时间当天就发了 offer

### 腾讯

学长帮忙直推了他实习的部门，所以很快就被捞起来了，等了三天来了一面

#### 一面

- 我们团队现在一个情况
- Vue、React 哪个更了解，说 React 的原理
- 项目中怎么用 node
- http

#### 二面

- 学前端遇到的最大困难

之后就等 hr 面，等 offer call 一直到现在...

### 百度

3.1 号在牛客上看到百度云上海组内直招，3.15 才联系我...

#### 一面

- 大部分也是基础问题
- forEach 和 map 有什么区别
- 懒加载的具体实现

#### 二面

- 牛客在线编程三道题：
  - 二分查找
  - 斐波那契数列
  - 输入两个整数 n 和 m，从数列1，2，3.......n 中随意取几个数,使其和等于 m ,要求将其中所有的可能组合列出来（背包问题）

#### 三面

三面没有问太多技术问题，主要就是确认能不能立刻入职

其实综合以上的面试也可以感觉到，掌握好网上大多数的面经在面试准备上基本就没什么问题了，规范的流程一般也是前一二面更多注重基础算法等等，后面就是看自身的沉淀了，这个就需要实际项目中有足够的技术深度学习，实践过对口的业务场景，平时多思考多总结，把自己的学习能力与潜力展示出来，大厂需要的就是这样的人。

同时对于实际的面试周期，也要有足够的耐心和毅力，面试本身是一个很耗心力的过程，一般做好准备再去应战，对于最后能不能拿到 offer 也有一定的运气成分，但我们能把握的就是扎实地提升自己，最后写一句之前看到的很有道理的话吧：大多数人总是高估自己一天能做的事，却低估自己一年能做的事。共勉！！！
